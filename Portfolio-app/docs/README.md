## 📞 문의 및 기여

### 문서 개선 제안

문서 개선 사항이 있다면:

1. GitHub Issue로 제안하기
2. Pull Request 직접 만들기
3. 팀 회의에서 논의하기

### 문서 작성 가이드

**마크다운 컨벤션:**
- 제목: `#` ~ `###` 사용
- 강조: `**굵게**`, `*기울임*`
- 코드: \`인라인\`, \`\`\`블록\`\`\`
- 이모지: 섹션 구분에만 사용
- 표: 정보 비교에 활용

**문서 구조:**
```markdown
# 제목

> 한 줄 설명

---

## 섹션 1

내용...

---

## 섹션 2

내용...

---

**문서 버전:** 1.0.0
**작성일:** YYYY-MM-DD
**관련 문서:** [링크](...)
```
